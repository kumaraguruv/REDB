post_multipart(host, selector, fields, files)

source code 

Post fields and files to an http host as multipart/form-data. fields is a sequence of (name, value) elements for regular form fields. files is a sequence of (name, filename, value) elements for data to be uploaded as files. Return the server's response page.

encode_multipart_formdata(fields, files)

source code 

fields is a sequence of (name, value) elements for regular form fields. files is a sequence of (name, filename, value) elements for data to be uploaded as files. Returns (content_type, body) ready for httplib.HTTP instance.
